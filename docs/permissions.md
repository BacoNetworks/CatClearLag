## Permissions

```
Whitelist item command - catclearlag.command.whitelistitem
Force unload chunks - catclearlag.command.unloadchunks
Generic re command - catclearlag.command.remove
	Remove hostile mobs - catclearlag.command.remove.hostile
	Remove all - catclearlag.command.remove.all
	Remove items - catclearlag.command.remove.items
	Remove living entities - catclearlag.command.remove.living
	Remove XP orbs - catclearlag.command.remove.xp
	Remove passive - catclearlag.command.remove.passive
	Remove named - catclearlag.command.remove.named
	Remove animal - catclearlag.command.remove.animal
Laggy chunks command - catclearlag.command.laggychunks
Force garbage collection - catclearlag.command.forcegc
```
